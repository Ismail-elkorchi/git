# Decisions

Architectural decisions MUST be recorded as ADR files under docs/decisions.
ADR files MUST use the template in docs/decisions/ADR-TEMPLATE.md.
ADR files MUST include a decision statement and consequences.
ADR files MUST include an Operational Rubric R section.
ADR files MUST include a Structured Disagreement section.
ADR files MUST include a Learning And Validation Loops section.
ADR files MUST include a Metrics And Evidence Controls section.
ADR files MUST include a Research Lenses section.
ADR files MUST include a Lens To Invariant Mapping section.
ADR files MUST include an Evidence Mapping section.
ADR files MUST include a Traceability Record section.
