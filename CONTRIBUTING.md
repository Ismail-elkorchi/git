# Contributing

All changes MUST be submitted as pull requests.
All pull requests MUST follow .github/pull_request_template.md.
All pull requests MUST pass node scripts/check.
All pull requests MUST keep one logical change scope.
All pull requests MUST include links to related issues or discussions when they exist.
All pull requests MUST NOT add new verification scripts.
All pull requests MUST update spec/state.yaml.
All contributions MUST be compatible with the MIT license in LICENSE.
